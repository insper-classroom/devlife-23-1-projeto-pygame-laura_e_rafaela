Sobre o jogo

História do Jogo

Dough boy, o biscoito de gengibre, está atrasado para a festa de natal da sua familia e para piorar sua situação ele prometou levar biscoitos para festa, mas ele esqueceu. Agora Dough boy conta com a sua ajuda para buscar os biscoito e atravessar a floresta gelada para chegar em casa! Mas tome cuidado, a floresta possui criaturas perigosas como o monstro da neve que, apesar, de fofo é letal.

Aprenda a jogar

Para jogar use as setas do teclado ou as letras A e D para poder ir para esquerda e direita respectivamente e a tecla espaço para pular. Você deve coletar os biscoitos para ganhar mais pontos,mas tome cuidado com os monstro de neve, eles podem te dar danos e tirar suas vidas!

Como posso jogar?

Primeiro é necessário baixar o VSCODE e que nele esteja instalado o PYGAME. Depois disso é preciso baixar os arquivos através do Zip code disponível abaixo. Assim que baixar o Zip Code nos dowloads, basta clicar nele para que esse se transforme em uma pasta e abrir essa pasta de arquivos no VSCODE. Finalmente agora é só entrar no Arquivo jogo_GM.py pelo VSCODE, dar play e se divertir!
